# F.R.I.D.A.Y

Персональная ИИ-ассистентка: сайт с регистрацией, бэк-офис для управления
подписками и Telegram-бот с движком на Claude.

Концепт — личная ассистентка в духе F.R.I.D.A.Y. из «Железного человека»:
забирает задачи, решает их сама, шлёт отчёты, делает напоминания.

## Структура (монорепозиторий)

```
FRIDAY/
├── apps/
│   └── web/          # Сайт + личный кабинет + бэк-офис (Next.js)
│   └── bot/          # Telegram-бот на grammY — этап 2
├── packages/         # Общий код — появится на этапе 2
├── supabase/
│   └── migrations/   # SQL-схема базы данных
└── docs/             # Архитектура и инструкция по запуску
```

## Быстрый старт

1. Прочитай **[docs/SETUP.md](docs/SETUP.md)** — что нужно создать (Supabase и др.).
2. `pnpm install`
3. Заполни `apps/web/.env.local` (шаблон — `.env.example`).
4. Примени SQL-схему из `supabase/migrations/` в своём Supabase-проекте.
5. `pnpm dev` → сайт на http://localhost:3000

## Дорожная карта

- **Этап 1 (текущий)** — сайт, регистрация, личный кабинет, бэк-офис, подписки.
- **Этап 2** — ядро Telegram-бота: привязка к владельцу по ID, движок Claude,
  базовые команды, напоминания.
- **Этап 3** — генерация файлов (PDF/DOCX/XLSX), картинок, оценка изображений,
  динамическое создание команд, автономные задачи, интеграции (Trello, Notion).
